Mot clef this dans un objet littéral

Attention ! Quand on place une fonction (méthode) dans une autre fonction (méthode), le mot clef this va pointer vers l'objet Global et donc this.radius = 20 ; va affecter à 20 un attribut radius appartenant à Global (Window)
Ceci est considéré comme un bug par nombre de personnes. Il en est ainsi pour tous les moteurs JS de tous les navigateurs car c'est dans le fonctionnement propre de JS.

Self pour palier au problème

Pour palier au problème, on déclare une variable self = this (qui peut s'appeler toto ou foo) dans la fonction ainsi ce n'est pas le mot clef this qui est utilisé mais self ce qui pare à ce "bug"
A priori, ce problème serait corrigé dans ECMAScript 6.
