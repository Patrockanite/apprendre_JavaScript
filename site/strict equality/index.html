Intérêt de l'égalité stricte en Javascript

Dans une structure conditionnelle de type if(x==y){
instruction ;
}
Javascript va retyper dynamiquement les variables s'il a besoin. Ainsi, si l'on affecte x ='4' et y = 4 de cette façon,
il va renvoyer True alors que x est un string et y est un int. Il commence par égaliser les types puis compare les valeurs

Pour interdire ceci on va utiliser l'égalité stricte avec ===
